Install

The latest version of Gren is 0.1.0. You can find the compiler binary for your OS at GitHub.

If you're interested in trying out the latest, though unstable, compiler then you can find builds of the main branch here.

Once you have the binary, you'll need to give it permission to execute and place it somewhere in your PATH. Below is an example for Mac OS.

 unzip gren-macOS.zip
 chmod +x gren
 mv gren /usr/local/bin/

You should now be able to execute the 'gren' command and see a friendly greeting.

If you don't know what to do next, check out the official language guide.
