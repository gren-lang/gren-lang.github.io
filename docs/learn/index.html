Learn

If you're new to the Gren programming language, the best way to learn is to read the book.

Once you understand the basics, you might want to take a look at a few example projects to get a better idea on the sort of things you can do.

You can read the documentation of the core packages, as well as other third party-packages at packages.gren-lang.org.

If you need any help along the way, or just want to chat with other people who're enthusiastic about Gren, then join our Discord.
